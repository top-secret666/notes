## 💻 Your answer (code / explanation)
- Код:

```java
import java.util.*;
public class MergeUtil {
    public static List<String> merge(List<String> a, List<String> b) {
        LinkedHashSet<String> set = new LinkedHashSet<>();
        set.addAll(a);
        set.addAll(b);
        return new ArrayList<>(set);
    }
}
```

- Объяснение: used LinkedHashSet to preserve insertion order and remove duplicates. Complexity O(n+m).
