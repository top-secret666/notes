## 🔌 Core Plugins
- Dataview: Query engine
- Templater: Automation
- QuickAdd: Fast note creation
- Tasks: Todo management
- Kanban: Project boards
- Advanced Tables: Data tables
- Style Settings: UI customization
- Git: Version control

---

## 🏷️ Metadata Schema
- <b>id:</b> Unique identifier
- <b>type:</b> Note type (lesson, project, etc.)
- <b>domain:</b> Knowledge domain
- <b>status:</b> Current status
- <b>level:</b> Difficulty/complexity
- <b>created/updated:</b> Timestamps
- <b>tags:</b> Categorization
- <b>related:</b> Connections
- <b>source:</b> Origin
- <b>review_cycle:</b> Maintenance schedule

## 🤖 Automation Rules
- Daily: Progress tracking
- Weekly: Review cycles
- Monthly: System audits
- CI: Validation on commits

## 🚀 Performance Targets
