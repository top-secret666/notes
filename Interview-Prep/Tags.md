# Tags

Центральная заметка для управления тегами в тетради.

## Основные теги
- #lesson — уроки
- #project — проекты
- #interview — подготовка к собеседованиям
- #resource — полезные материалы
- #idea — идеи и заметки

## Рекомендации
- Используйте теги для быстрого поиска и фильтрации
- Не перегружайте заметки большим количеством тегов
- Для новых категорий создавайте отдельные теги
