# Plugins Install Guide

Руководство по установке и настройке рекомендуемых плагинов для продуктивной работы в Obsidian.

## Рекомендуемые плагины
- Dataview — для динамических списков и автоматизации
- Kanban — визуальное управление задачами
- Templater — шаблоны и автоматизация
- Excalidraw — визуальные заметки и схемы
- Calendar — календарь и планирование
- Advanced Tables — удобные таблицы

## Как установить
1. Откройте настройки Obsidian → Community plugins
2. Найдите нужный плагин по названию
3. Установите и активируйте
