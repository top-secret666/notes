
# Plugin Settings

**Внимание!** Для ускорения работы Obsidian оставлены только самые нужные плагины:
- Dataview
- Templater
- Kanban
- Excalidraw
- Calendar
- Tasks
- Style Settings

Остальные плагины отключены или удалены для повышения производительности.

Настройки и рекомендации по плагинам для Obsidian.

## Автоматизация
- [[07_AI_ADMIN/Automation-readme|Automation Readme]]

## Основные плагины
- Dataview: включён, используется для автоматических списков
- Kanban: для визуального управления задачами
- Templater: шаблоны для заметок и уроков

## Советы
- Регулярно обновляйте плагины
- Храните резервные копии настроек
