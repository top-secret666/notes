Аннотации PDF и работа с пером — руководство

Цель: быстро настроить рабочий поток для рукописных заметок и аннотирования PDF в Obsidian и внешних приложениях.

1) Установка плагинов (в Obsidian)
- Settings → Community plugins → выключить Safe Mode → Browse → установить:
  - `Excalidraw` — для рукописных заметок/рисунков.
  - `Canvas` — холсты с поддержкой pen (если установлен в твоей версии).
  - `Annotator` / `PDF Highlighter` (поиск по названию; в разных версиях плагин может называться по‑разному) — для аннотаций PDF.

2) Примеры использования
- Рукописная заметка в Excalidraw:
  - Создай файл: `New note` → `Insert Excalidraw` или создай файл с расширением `.excalidraw`.
  - Выбери Pen tool и пиши пером прямо в файле. Сохраняй внутри vault.
- Встраивание Excalidraw в Markdown:
  - Вставь ссылку на файл: `![[Example-Excalidraw.excalidraw]]` — он отобразится прямо в заметке.

3) Аннотация PDF
A) В Obsidian через плагин (если доступен):
  - Открой PDF в Obsidian → включи режим аннотаций в панели плагина → используй Pen/Highlighter/Notes → Save.
  - Сохранённые аннотации могут быть встроены в файл или экспортированы.
B) Внешние приложения (если плагин неудобен):
  - Microsoft Edge (Windows) — Open PDF → Draw → сохраняй изменения прямо в файле.
  - Xodo / Foxit / Adobe Acrobat Reader — все поддерживают inking и сохранение аннотаций.
  - Drawboard (если есть) — платный, очень удобный для рукописных заметок.

4) Рабочий пример (что я создал)
- Файл `Example-Excalidraw.excalidraw` — пример холста, который можно встроить в заметку через `![[Example-Excalidraw.excalidraw]]`.
- Папка `media/` — где хранить экспортированные gif/png из Excalidraw или записи демонстраций.

5) Советы по ergonomics
- В драйвере планшета назначь кнопку Erase и одну кнопку для Undo (если есть).
- Используй 1–2 стили пера: толстый для заметок, тонкий для пометок.
- Для конвертации рукописи в текст: OneNote → копировать как текст, или Windows Handwriting Panel.
