# PDF++ — настройки под планшет (киберпанк)

#obsidian #pdf

## Режимы и поведение
- Контекстное меню: всегда при отпускании пера/мыши
- Включено редактирование PDF (аннотации)
- Боковая панель: справа, не скрывается автоматически

## Быстрые действия (hotkeys)
Настрой в Settings -> Hotkeys:
- PDF++: Toggle PDF edit
- PDF++: Toggle sidebar
- PDF++: Next page / Previous page
- PDF++: Zoom in / Zoom out
- PDF++: Fit page width

## Жесты/перо
- Используй выделение маркером + быстрые цвета (палитра сверху)
- Для конспекта: Yellow для базовых фактов, Note для терминов, Important для определений


## Рекомендации
- Используйте горячие клавиши для ускорения работы с PDF
- Для аннотирования выделяйте цветами по смыслу
- Храните важные конспекты в отдельной папке

## Подсказка
Если контекстное меню мешает, можно вернуть режим "Mod" (только с клавишей).
